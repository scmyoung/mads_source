1.4. Obtaining Support¶

If this documentation does not address your question/problem and you need support, please feel free to post in our iPhone Developers Forum.

When posting, please:

  • Check the Frequently Asked Questions (FAQ) and the rest of this documentation first.
  • Start a new thread for a new question - do not “hijack” an unrelated thread.
  • Post complete details of your problem - complete error messages as well as relevant source code and images.
  • Log-in to receive email, or check back within a day or so - you will almost always get a response.

For the latest support information, please see http://zbar.sf.net/iphone
